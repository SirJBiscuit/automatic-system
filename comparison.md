Before installing pterodactyl panel:
1. Install dependencies
2. Download files
3. Open .env in nano
4. Find and edit PTERODACTYL_URL
5. Find and edit PTERODACTYL_API_KEY
6. Find and edit WEB_USERNAME
7. Find and edit WEB_PASSWORD
8. Find and edit SECRET_KEY
9. Save and exit
10. Hope you didn't make typos


With the installer:
1. Run installer
2. Answer 4 simple questions
3. Done! ✅
