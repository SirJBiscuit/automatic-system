## 🎮 **SERVER GROUPS**

### **How It Works**

The system automatically categorizes servers based on their names:

#### **Minecraft** 🎮
- Detects: "minecraft", "mc"
- Example: "MC Survival", "Minecraft Creative"

#### **Rust** 🔫
- Detects: "rust"
- Example: "Rust PvP", "Rust Modded"

#### **ARK** 🦖
- Detects: "ark"
- Example: "ARK Survival", "ARK PvE"

#### **Valheim** ⚔️
- Detects: "valheim"
- Example: "Valheim Server", "Valheim Co-op"

#### **Other** 📦
- Everything that doesn't match above
- Example: "Terraria", "7 Days to Die"

### **Usage**
1. Use the **Group Filter** dropdown in search bar
2. Select a game type
3. Only servers matching that type will show
4. Combine with search and status filters
